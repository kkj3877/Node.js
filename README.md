# Node.js
- Node.js 는 확장성 있는 네트워크 어플리케이션(특히 서버 사이드) 개발에 사용되는 소프트웨어 플랫폼이다.
- JS 를 사용하여 논블로킹(Non-blocking) I/O 와 단일스레드 이벤트 루프를 통해 높은 처리 성능을 낼 수 있다.
- 내장 HTTP Server 라이브러리를 포함하고 있어 아파치 등의 별도의 소프트웨어 없이 동작 가능하다.
